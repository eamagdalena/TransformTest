 How to play: 
 
 1. Run Main.java
 2. Open in a browser:  http://localhost:8080/static
 3. Have fun
  
 What it contains:
 
 * A JAX-RS Webservice with 2 methods
 * A simple web page that uses these services
 * A DAO Layer that keeps the game state for every player (GameDAO)
 * A Game model and logic (model package)
 * An "AI"
 * Grizzly for ease of use (Main.java)
